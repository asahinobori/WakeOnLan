# Wake up PC
# About
**1) Wake up PC through network**  
**2) Custom target host, port and mac**  
  
# Usage
```
wol [-host -mac -port] | [-h]
  -host string
        host domain or broadcast address (default "your.domain.org")
  -mac string
        mac address, ex:BB:CC:DD:EE:FF:11 (default "AA:BB:CC:DD:EE:FF")
  -port int
        host port (default 9)
```
